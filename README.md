# Proyectos Tecnicatura - PowerSystem2024

Este repositorio reúne proyectos en los que participé como parte del 
equipo **Solución 202**, durante la Tecnicatura en Programación (UTN 
Facultad Regional San Rafael), dentro de la organización PowerSystem2024.

Los proyectos se desarrollaron con **metodología ágil (Scrum)**, en 
equipos de trabajo, con entregas iterativas por sprint.
